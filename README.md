# להיות שם באמת – Basic-Understanding-Life

מפתח: **Michael Papismedov – MP**

## תיאור כללי

זהו אתר הסבר רגשי־אנושי (Single Page Application ב־React) שעוסק בנוכחות, תמיכה ותגובה נכונה ברגעים פיזיים ורגשיים קשים במערכת יחסים.

האתר נבנה לפי עקרונות:
- נוכחות כפעולה ולא רק כמילה.
- הסבר רגיש לשני הצדדים בקשר – הצד שנפגע והצד שלא תמיד מבין מה קרה.
- חוויית קריאה ארוכה, נעימה ולא מציפה.

## טכנולוגיה והרצה

- React + TypeScript
- Vite

להתקנת התלויות והרצת Build:

```bash
npm install
npm run build
```

להרצת סביבת פיתוח:

```bash
npm run dev
```

## מבנה האתר

- Single Page Application (SPA) מבוסס סקשנים:
  - אזור פתיחה (Hero) – כותרת ראשית ומשפט רגשי מרכזי.
  - סקשני הסבר על נוכחות, גוף/נפש, כעס, טעויות, מה נכון לעשות, דפוסים וצרכים רגשיים.
  - משחקון אינטראקטיבי קצר.
  - אזור שאלות ותשובות (FAQ).
  - מסר סיום רגשי.

## פיצ׳רים מרכזיים

- מערכת חידון אינטראקטיבי:
  - 5 שאלות סגורות + שאלה פתוחה לסיכום אישי.
  - שמירת כל מצב החידון באחסון מקומי (localStorage), כולל:
    - נוסח כל שאלה.
    - כל אפשרויות התשובה.
    - התשובה שנבחרה לכל שאלה.
    - הטקסט המלא של השאלה הפתוחה.
- שאלה פתוחה בסיום:
  - מאפשרת למשתמשת לכתוב באופן חופשי על תחושותיה בתוך הקשר.
  - נשמרת מקומית בדפדפן להמשך.
- מצב תצוגה בהיר וכהה:
  - כפתור החלפה בהדר בין Light / Dark.
  - שמירת הבחירה ב-localStorage והחלתה אוטומטית בכניסה הבאה.
- רספונסיביות מלאה:
  - מותאם למובייל, טאבלט ודסקטופ, כולל מכשירים כמו iPhone 16 Pro / Pro Max ו-Samsung Galaxy S22 Ultra.
- תפריט מובייל מסוג "המבורגר":
  - פתיחה וסגירה רכה של תפריט הניווט במובייל.
  - מעבר גלילה חלק לסקשנים שונים בעמוד.
- אנימציות מעבר עדינות:
  - hover רך על כפתורים.
  - מעבר חלק בין מצבי התצוגה.

## עקרונות רספונסיביות

- עיצוב רספונסיבי מלא למובייל, טאבלט ודסקטופ.
- תשומת לב מיוחדת לחוויית קריאה במכשירים:
