# להיות שם באמת – Basic-Understanding-Life

מפתח: **Michael Papismedov – MP**

## תיאור כללי

זהו אתר הסבר רגשי־אנושי (Single Page Application ב־React) שעוסק בנוכחות, תמיכה ותגובה נכונה ברגעים פיזיים ורגשיים קשים במערכת יחסים.

האתר נבנה לפי עקרונות:
- נוכחות כפעולה ולא רק כמילה.
- הסבר רגיש לשני הצדדים בקשר – הצד שנפגע והצד שלא תמיד מבין מה קרה.
- חוויית קריאה ארוכה, נעימה ולא מציפה.

## טכנולוגיה והרצה

- React + TypeScript
- Vite

להתקנת התלויות והרצת Build:

```bash
npm install
npm run build
```

להרצת סביבת פיתוח:

```bash
npm run dev
```

- עיצוב רספונסיבי מלא למובייל, טאבלט ודסקטופ.
